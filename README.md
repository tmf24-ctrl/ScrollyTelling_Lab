# Scrollytelling Spec-Driven Test Project

A small test project that applies scrollytelling plus spec-driven development.

## Live Site

- GitHub Pages: `https://tmf24-ctrl.github.io/ScrollyTelling_Lab/`

Enable GitHub Pages in repository settings to publish this URL.

## Repository

- GitHub Repo: `https://github.com/tmf24-ctrl/ScrollyTelling_Lab`

## What This Includes

- Scrollytelling homepage with sticky stage and step-based narrative.
- Two linked content pages:
  - `why-this-project.html`
  - `process-notes.html`
- Return-link query flow (`returnTo`) between branch pages and homepage.
- Spec-driven project artifacts:
  - `docs/specs/`
  - `docs/phases/`

## Run Locally

Open `index.html` directly, or use a simple local server if preferred.

Example with PowerShell (Python installed):

```powershell
python -m http.server 5500
```

Then visit `http://localhost:5500`.
