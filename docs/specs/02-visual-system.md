# Visual System

## Typography

- Display: Fraunces.
- Body/UI: Space Grotesk.

## Style Direction

- Warm editorial palette with earthy accents.
- Layered gradients and soft ambient shapes.
- Rounded cards and thin lines for structure.

## Responsiveness

- Desktop: split scrolly layout (sticky stage + narrative steps).
- Mobile: stack stage above content with reduced spacing.
