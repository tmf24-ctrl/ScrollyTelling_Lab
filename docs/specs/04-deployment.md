# Deployment

## Target

- GitHub Pages static hosting.

## Structure

- Root-level static files (`index.html`, CSS, JS, content pages).
- No build step required.

## Publish Steps

1. Push repo to GitHub.
2. In repository settings, enable Pages from `main` branch root.
3. Confirm site URL pattern: `https://<username>.github.io/<repo>/`.
4. Add hosted URL in README.
