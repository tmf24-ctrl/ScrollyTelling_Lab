# Information Architecture

## Pages

- `index.html`: scrollytelling homepage with sticky stage and 5 narrative slides.
- `why-this-project.html`: regular content page explaining rationale and outcomes.
- `process-notes.html`: regular content page describing control loop and execution notes.

## Navigation

- Top navigation from homepage to both content pages.
- Content pages include a `returnTo` query param to return users to a homepage anchor.

## Core User Path

1. Land on homepage.
2. Scroll through the narrative.
3. Branch to content pages for detail.
4. Return to homepage anchor for continuity.
