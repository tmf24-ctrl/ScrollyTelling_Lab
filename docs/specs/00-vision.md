# Vision

Build a small, portfolio-style scrollytelling site that demonstrates a story-first homepage and linked regular content pages.

## Objectives

- Tell a clear narrative through scroll progression.
- Keep interaction lightweight and reliable for GitHub Pages.
- Show spec-driven development via concise docs/specs and docs/phases artifacts.
- Provide links in README for repository and hosted Pages site.

## Non-Goals

- Full CMS integration.
- Backend APIs.
- Advanced animation libraries.
